# Build Study Tutorial

Welcome to the Build Study page of OpenClinica! This page is designed to help you build and configure your Study by following a task-based approach. 

The tasks listed need to be completed in order to start using your Study. Once you have finished each task, check "Mark Complete" then click the "Save" button. 

* To start a task, select the "Create" icon. You can always select the "Create" icon to add a new component to your Study.
* Select the "Edit" icon to edit the values for a task.
* Select the "View" icon to view the specified values for that task.
